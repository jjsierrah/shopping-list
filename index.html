🛒 JJ Shopping List

    Ayuda - JJ Shopping List

    Versión: v3.9

    Aplicación PWA para gestionar tu lista de la compra con las siguientes funcionalidades:


    • ✅ Añadir productos con categoría y ubicación
    • ⭐ Marcar productos como favoritos
    • 📌 Marcar productos como predeterminados
    • 🔄 Gestionar categorías y ubicaciones
    • 📋 Copiar lista al portapapeles
    • 📱 Instalable como aplicación en Android
    • 🔒 Datos guardados localmente (sin conexión necesaria)
    • ↕️ Reordenar productos con arrastrar y soltar
